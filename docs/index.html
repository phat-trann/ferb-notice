Chrome MV3 extension

Check in, work enough, check out.

FerbNotice reminds you to check in once per day, tracks your configured work duration, and shows the checkout reminder exactly when your rounded work window is complete.

Why it exists

A small extension for one daily workflow.

Daily reminder

The first eligible active tab of each local day triggers a check-in reminder modal with an editable check-in time.

Configurable schedule

Adjust the latest on-time check-in threshold, work duration, and checkout rounding slot from the setup popup.

Manual correction

If the browser opens late, correct or clear today's recorded check-in data and FerbNotice recomputes the due time and badge state.

Production build

Create a release-ready extension package.

source ~/.nvm/nvm.sh
nvm use node
npm ci
npm run pack:production

The production package is generated as ferbnotice-production.zip. The unpacked extension files are generated in dist.
